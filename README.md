A minimal Dockerfile based on Node.js 24 LTS (Krypton) alpine with Firefox installed.

## What's included

- Node.js 24 LTS
- npm 11
- yarn
- Firefox

### Available platforms

These Dockerfiles leverage the new `buildx` functionality and offer the following platforms:

- linux/amd64
- linux/arm64
